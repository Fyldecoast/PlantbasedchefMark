Performance and  Optimisation
The site was built for speed and reliability, achieving a **100/100 performance score** on desktop.
- **Vanilla JavaScript:** Zero-dependency interaction logic.
- **Tailwind CSS:** Optimised utility first styling.
- **Formspree:** Robust enquiry handling.
- **Lazy Loading:** All high-resolution assets are optimised for network efficiency.
